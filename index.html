Firebase Access Code Generator

This is a tool to generate secure, one-time access codes and store them in Firebase Firestore.

Implementation Options

Option 1: iFrame Embed

Add this code to your funnel:

<iframe src="https://chrolloace.github.io/capaiembedcode/access-code-iframe.html?userId=USER_ID&purchaseId=ORDER_ID" style="border:none; width:100%; height:250px;"> </iframe>

Option 2: Direct Script Embed

Add these scripts to your funnel:

<script src="https://www.gstatic.com/firebasejs/9.23.0/firebase-app-compat.js"></script> <script src="https://www.gstatic.com/firebasejs/9.23.0/firebase-firestore-compat.js"></script> <script src="https://chrolloace.github.io/capaiembedcode/access-code-embed.js"></script> <div id="access-code-container"></div> <script> new AccessCodeGenerator({ containerId: 'access-code-container', firebaseConfig: { apiKey: "AIzaSyDnSewoxH75cYRPlzjX8SS8SseLTakMl_8", authDomain: "capped-ai.firebaseapp.com", projectId: "capped-ai", storageBucket: "capped-ai.appspot.com", messagingSenderId: "897068191530", appId: "1:897068191530:web:7e3d4af4908ce717021853" }, userId: "USER_ID", purchaseId: "ORDER_ID" }).init(); </script>
